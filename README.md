# Plastik Köşebent Web Sitesi

Bu proje, plastik köşebent ürünleri için profesyonel bir web sitesidir. Modern tasarım, responsive yapı ve kullanıcı dostu arayüz ile müşterilere kapsamlı ürün bilgileri sunmaktadır.

## Özellikler

- 6 ana sayfa (Ana Sayfa, Ürünler, Uygulamalar, Katalog, Hakkımızda, İletişim)
- Detaylı ürün sayfaları (10x10, 15x15, 20x20, 25x25, 30x30, 40x40, 50x50, 75x75 mm)
- Responsive tasarım (mobil uyumlu)
- Modern CSS ile şık arayüz
- Etkileşimli JavaScript özellikleri
- Ürün galerileri ve filtreleme
- İletişim formu
- SEO uyumlu yapı

## GitHub Pages Yayını

Bu web sitesi GitHub Pages üzerinden yayınlanabilir:

1. Bu depoyu GitHub hesabınıza klonlayın
2. Değişiklikleri yapın ve commit edin
3. GitHub Pages ayarlarından ana dalı (main/master) yayınlayın
4. Web sitesiniz `https://kullaniciadiniz.github.io/plastik-kosebent/` adresinde yayınlanacak

## Teknolojiler

- HTML5
- CSS3 (Responsive)
- JavaScript (ES6+)
- Google Fonts (Poppins)
- Picsum Photos (Resimler için)

## Kullanım

Web sitesini yerel olarak çalıştırmak için:

1. Dosyaları bilgisayarınıza indirin
2. `index.html` dosyasını tarayıcıda açın
3. Web sitesini yerel olarak görüntüleyin
